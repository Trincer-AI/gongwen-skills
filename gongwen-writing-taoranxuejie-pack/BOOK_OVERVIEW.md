# 公文写作技能包 — 整书理解 (Adler Analysis)

## 基本信息

- **书名**: 《从零开始学公文写作》+ 《公文写作大手笔：从小笔头到笔杆子》
- **作者**: 陶然学姐
- **出版年**: 2023 / 2024
- **出版社**: 北京大学出版社
- **总字数**: 约800KB（两书合计）

---

## 步骤1 — 结构 (Structural)

### 两书定位

两书是姐妹篇：
- **第一本**：《从零开始学公文写作》(2023) — 入门篇，从零开始讲公文写作的痛点、方法和技巧
- **第二本**：《公文写作大手笔》(2024) — 进阶篇，深化思维逻辑，讲解深度检索和办文办事

### 主旨（一句话）

> 公文写作是职场人"被看见"的能力通道——它不是天赋，而是可以通过系统学习和刻意练习掌握的实战技能。

### 主要部分及关系

**第一本骨架（8章）**：

```
第1章 职场新人写材料有多难（痛点识别）
    ↓
第2章 写材料的"坑"（格式/审题/博好感）
    ↓
第3章 痛定思痛，从零学起（核心方法论：句/标题/结构/内容）
    ↓
第4章 学得自信满满，现实骨感（理论与实践的Gap）
    ↓
第5章 七个技巧（美人/减法/复核/换位/整理/漏洞/参照）
    ↓
第6章 长篇材料撰写（讲话稿/总结/述职报告/调研/先进事迹）
    ↓
第7章 机关呈現与过程管理
    ↓
第8章 法定文种使用误区
```

**第二本骨架（章节主题）**：

```
深度检索方法论（检索思维是基础）
    ↓
思维逻辑深化（主观能动性/换位/系统思维）
    ↓
办文办事细节（容易被忽视的软技能）
    ↓
各类长文种深化（讲话稿/总结/述职报告）
```

### 作者试图解决的核心问题

1. **入门难**：职场新人不知道公文写作的标准是什么、怎样才算达标
2. **提升慢**：写了很多篇但水平提高有限（量变没有引起质变）
3. **长文怕**：领导讲话、工作总结等长篇材料无从下手
4. **检索弱**：不会高效检索高质量素材
5. **思维浅**：缺乏系统性思维和站位意识

## 步骤2 — 解释 (Interpretive)

### 关键术语词典

| 术语 | 陶然学姐的用法 |
|---|---|
| **兵器库** | 指公文写作者积累的素材库（包括上级文件、兄弟单位材料、领导批注等） |
| **美人技巧** | 鉴别好材料、学习好材料的方法——不是抄，而是分析好在哪里 |
| **减法技巧** | 删除仍通顺的部分，保留核心；公文要精简，不是堆砌 |
| **换位技巧** | 站在领导/上级的位置思考，而不是只从自己的角度写 |
| **虚实结合** | 公文中"虚"（理论高度）与"实"（具体举措）要配合，不能全是虚话 |
| **量变到质变** | 写作能力的提升需要刻意练习+反馈+复盘，单纯重复不会进步 |
| **葵花宝典** | 作者幽默指代公文写作的核心方法论（第5章七个技巧） |
| **主动思维** | vs "等靠要"——不被动等领导指示，而是主动检索、主动思考 |
| **主观能动性** | 面对新任务时不抗拒、不推脱，转化为"我学学、查查看" |
| **深度检索** | 不是简单的百度，而是多种思维方式的组合检索（同义词/动宾短语/金句拆解） |
| **基层到机关** | 文风从"花拳绣腿"到"扎实用料"的转变要求 |

### 核心命题

1. **公文写作是职场核心竞争力** — 提笔能写、开口能讲是领导对年轻人的基本要求
2. **公文写作可以学习** — 不是天赋，而是方法论+刻意练习
3. **检索能力是基础** — 充分占有资料才能写好，公文写作赢在起跑线
4. **量变要引起质变需要刻意练习** — 重复写而不复盘等于没写
5. **七个技巧是写好公文的关键** — 美人/减法/复核/换位/整理/漏洞/参照
6. **虚实结合决定文章高度** — 全虚是流水账，全实是没高度
7. **格式错误是刚性错误** — 犯一次可能几年无缘晋升
8. **主动思维vs等靠要是成败分水岭** — 面对新任务的心态决定成长速度

## 步骤3 — 批判 (Critical)

### 作者的时代局限

1. **技术背景**：书写于2023-2024年，部分检索方法（如AI辅助写作）尚未普及，现在已有变化
2. **体制内视角**：作者长期在体制内，部分方法对民营企业不完全适用

### 作者的立场盲点

1. **成功者偏差**：作者是"十年公文写作"的成功者，容易低估初学者的实际困难
2. **大城市视角**：案例多来自省市级机关，乡镇基层的情况可能不同
3. **女性视角**：作为女性写作者，某些场景（如被大领导当众表扬）的描述可能不同

### 未被证明的假设

1. "写作是可以教的" — 作者假设所有技巧都可以拆解，但承认有些只能靠"功底，慢慢来"
2. "只要努力就能写好" — 没有讨论能力天花板问题
3. "公文写作是进步最快的路径" — 这是普遍说法，但作者没有提供数据支持

### 反对意见

1. **"写得太好会被拴住"** — 很多人怕因为写得好而被一直留在写作岗，失去其他发展机会
2. **"AI时代公文写作不再重要"** — 未来可能AI辅助写作，人工写作的价值在哪里？
3. **"写作能力不等于综合能力"** — 写得好不代表能当好领导

## 步骤4 — 应用潜力 (Applicability)

### 可以skill化的内容

| 类型 | 候选skill | 优先级 |
|---|---|---|
| **方法类** | 深度检索法、金句收藏法、标题设计法、句子撰写法、结构构建法 | ★★★★★ |
| **方法类** | 内容扩充法、减法写作、复核技巧、换位技巧 | ★★★★☆ |
| **方法类** | 材料转化法（同一内容五种写法）、讲话稿撰写、总结撰写 | ★★★★☆ |
| **知识类** | 公文格式（页面/字体/标点）、15种法定文种、标题模式库 | ★★★★★ |
| **知识类** | 易错字词辨析、问题分析框架 | ★★★☆☆ |
| **概念类** | 主动思维vs等靠要、主观能动性激活、虚实结合 | ★★★★★ |
| **概念类** | 量变到质变、写作价值创造 | ★★★★☆ |

### 不适合skill化的

- 纯情感描述（如作者的写作心路历程）→ 可作为案例素材但不是skill
- 无法拆解的"手感"类内容 → 承认"没办法取巧，就是功底"

### 预估skill数量

- **方法类**：15-20个
- **知识类**：8-10个
- **概念类**：5-8个
- **合计约**：30-40个skill

## 技能包设计说明

本技能包设计为**三个角色合一**：
1. **秘书**：直接帮用户写各类公文（通知/报告/总结/讲话稿等）
2. **思维顾问**：提供公文写作的思路、框架、方法指导
3. **角色扮演**：扮演上级领导，帮忙批注、修改、预演

每个skill的A2 trigger需覆盖三个场景的语言信号。
