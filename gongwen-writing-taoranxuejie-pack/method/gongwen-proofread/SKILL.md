# 复核技巧

## R — 原文

"切忌对着电脑用眼睛检查稿件，这种方式会'坑'到你。因为用眼睛浏览屏幕存在跳跃性...正确的复核步骤：1.将稿件打印为纸质文档；2.拿出铅笔修改；3.出声音有感情地朗读；4.导出PDF版本到手机；5.隔天再复核。"

## I — 方法论骨架

**五步复核法** = 打印纸质 → 铅笔标记 → 出声朗读 → PDF手机看 → 隔天再看

复核vs屏幕浏览：屏幕浏览存在跳跃性，手眼分离容易漏错

工具选择：铅笔（灰色能和黑白打印文字区分，可擦）

注意：慎用朗读软件转语音，存在泄密风险。

## A1 — 书中的应用

五步详解：
1. 打印纸质：手眼分离 vs 屏幕的跳跃性
2. 铅笔标记：灰色能区分黑白打印文字，可擦便于修改
3. 出声朗读：降低跳字概率，发现拗口处
4. PDF手机看：小屏全貌减少遗漏
5. 隔天再看：大脑休息后更清醒，能发现问题

慎用朗读软件：存在泄密风险，内部材料内容不宜上传

## A2 — 触发场景

**秘书场景**："稿子写完了，帮我检查一下有没有错"

**思维顾问场景**："为什么打印出来比在电脑上看更容易发现错误？有什么系统复核方法？"

**角色扮演场景**："假设你是领导，做最后的审稿批注"

## E — 可执行步骤

### Step 0 - 预检

- [ ] 确认是终稿阶段，不是正在修改中
- [ ] 检查是否涉及敏感内容（涉密内容不能使用朗读软件）
- [ ] 确认有打印机可用

### Step 1 - 打印纸质

- [ ] 将文档打印为纸质版本
- [ ] 准备铅笔（灰色）用于标记
- [ ] 确保打印清晰、字号适宜阅读

### Step 2 - 铅笔标记

- [ ] 逐段阅读，用铅笔标记问题处
- [ ] 灰色标记与黑白文字区分度高
- [ ] 可擦特性便于修改后重新标记

### Step 3 - 出声朗读

- [ ] 有感情地朗读全文
- [ ] 降低跳字概率，发现拗口处
- [ ] 注意标点停顿，检验语气是否合适

### Step 4 - PDF手机复核

- [ ] 导出PDF版本到手机
- [ ] 在小屏上看全貌，减少遗漏
- [ ] 检查排版问题

### Step 5 - 隔天再看

- [ ] 大脑休息后更清醒
- [ ] 能发现之前忽略的问题
- [ ] 最终确认无误后交稿

## B — 边界与Fallback

### 边界条件表

| 边界类型 | 触发条件 | 处理动作 |
|----------|----------|----------|
| 阶段边界 | 还在撰写过程中 | 暂停复核，完成撰写后再复核 |
| 时间边界 | 时间紧迫无法完整复核 | 至少完成 Step 1-3（打印+标记+朗读） |
| 工具边界 | 没有打印机可用 | 使用阅读模式强制逐字阅读 |
| 安全边界 | 涉密内容 | 禁用在线朗读软件，使用本地复核 |
| 重大问题 | 复核发现重大问题 | 回到撰写阶段修改，不强行交稿 |

### Fallback策略表

| 困境 | 应对策略 | 操作要点 |
|------|----------|----------|
| 没有打印机 | 使用阅读模式 | 强制自己逐字阅读，手眼分离 |
| 时间紧迫 | 至少完成前三步 | 打印+标记+朗读不能省 |
| 发现重大问题 | 回退修改 | 不强行交稿，先修改再复核 |
| 无法隔天 | 跳过Step 5 | 至少完成Step 1-4 |
| 朗读容易跳字 | 用笔指着读 | 强制逐字，防止跳跃 |

**核心原则**：复核是交稿前最后关卡，屏幕浏览无法替代打印复核，涉密内容禁用在线工具。
