# 材料转化技巧

## R — 原文

"同一个内容，如何翻出五种花样？用做菜比喻，就是同一种食材，你要反复烹饪，清蒸、醋熘、红烧……翻出各种花样。"

## I — 方法论骨架

**同内容五级升级法** = 上网信息 → 工作简报 → 工作总结 → 领导讲话 → 工作思路

升级本质：时态从完成时变将来时（用"要"字），加入"继续"等词推进工作纵深，语言风格从客观描述变为主观要求。

## A1 — 书中的应用

五级示例（以同一工作为例）：
1. 上网信息 = 要素捋顺（帽段+举措+成效）
2. 工作简报 = 充实内容 + 提炼标题
3. 工作总结 = 提炼 + 压缩 + 提高站位
4. 领导讲话 = 转换语言风格 + 变换为意义/原因/影响/要求
5. 工作思路 = 深化推广 + 调整时态

升级关键：时态调整（完成时→将来时），词汇调整（加入"要""继续"）

## A2 — 触发场景

**秘书场景**："我手头有一份工作总结，能不能改成领导讲话稿"

**思维顾问场景**："同样的工作内容，怎么做到一鱼五吃？有什么文种转换技巧？"

**角色扮演场景**："假设你是领导，要求把这篇简报升级成工作思路"

## E — 可执行步骤

### Step 0 - 预检

**Input**: 原始素材（任意文种）
**Output**: 判断结果（是否可升级 + 目标文种）

- [ ] "素材完整度≥60%吗？"否 → 回到 fw-009 内容扩充法补充
- [ ] "目标文种比当前文种高阶吗？"否 → 终止（本技能仅适用升级场景）
- [ ] "素材关键要素（背景/举措/成效）是否齐全？"否 → 先补充再升级

### Step 1 - 分析现有素材

**Input**: 原始素材
**Output**: 素材分析表（当前等级 + 可复用要素清单）

- [ ] "梳理出几个关键要素？"（目标≥3个：背景/举措/成效）
- [ ] "当前素材属于五级中的哪一级？"（上网信息/简报/总结/讲话/思路）
- [ ] "可复用的核心内容有几段？"（目标≥1段可复用）
- [ ] "时态是完成时还是将来时？"（记录原时态，作为转换依据）

**质量检查点**：
- [ ] "删除这段后，核心逻辑断裂吗？"断裂 → 保留
- [ ] "这段话是客观描述还是主观要求？"客观→需转换风格

### Step 2 - 确定升级路径

**Input**: 素材分析表
**Output**: 升级路径选择

| 当前文种 | 目标文种 | 转换关键词 | 时态调整 |
|---|---|---|---|
| 上网信息 | 工作简报 | 充实内容+提炼标题 | 完成时→完成时 |
| 工作简报 | 工作总结 | 提炼+压缩+提高站位 | 完成时→完成时 |
| 工作总结 | 领导讲话 | 转换风格+意义/原因/影响/要求 | 完成时→将来时 |
| 领导讲话 | 工作思路 | 深化推广+调整时态 | 将来时→将来时 |

- [ ] "升级路径在上表范围内吗？"否 → 降低目标或拆分升级
- [ ] "目标文种需要几级升级？"（>1级则需分步执行）

### Step 3 - 执行转换

**Input**: 原始素材 + 升级路径
**Output**: 转换后初稿

- [ ] "时态已调整为'要''继续'等将来时态词？"（目标≥3处）
- [ ] "语言风格从客观描述变为主观要求？"（目标≥2处"必须""应当"）
- [ ] "缺失要素已补充？"（对照目标文种结构检查）
- [ ] "标题已按目标文种风格重写？"（符合该文种命名规范）

**质量检查点**：
- [ ] "删除'要''继续'后，句意是否改变？"改变→这些词是关键，不要删
- [ ] "是否存在上级对下级的语气？"是→改为平级或上级对下级

### Step 4 - 校验

**Input**: 转换后初稿
**Output**: 最终稿 + 校验报告

- [ ] "通读一遍，逻辑连贯吗？"（卡顿处标记待改）
- [ ] "核心信息失真率≤10%吗？"否 → 回退 Step 3 重写
- [ ] "风格符合目标文种吗？"（对照该文种历史公文检查）
- [ ] "字数在目标文种合理范围内？"（简报500-800字，总结1500-3000字，讲话2000-5000字）

## B — 边界与Fallback

**边界**（场景|触发条件|处理动作）：

| 场景 | 触发条件 | 处理动作 |
|---|---|---|
| 素材不足 | 完整度<60% | 回到 fw-009 内容扩充法补充后再升级 |
| 越级跳跃 | 跨越2个以上文种 | 分步执行，每次升一级 |
| 风格不匹配 | 转换后读起来不像该文种 | 参照目标文种历史公文重新调整 |
| 内容不支持 | 素材无法支撑目标文种要求 | 降低目标文种或回到 fw-009 补充 |
| 时间紧迫 | 紧急材料 | 先完成低阶版本，后续再升级 |

**Fallback**：
- 素材不足以支撑目标文种 → 回到 fw-009 内容扩充法补充
- 转换后风格不对 → 参照目标文种的历史公文重新调整
- 不确定能否升级 → 从低级开始逐步升级尝试
