# 理论文章写作方向区分法

**R（Reading）— 原文引用**
> 理论文章有两种方向：学者向（讲道理学理哲理）和领导向（谈认识要求举措）。

**I（Interpretation）— 理解重写**
理论文章的写作方向有学者向和领导向之分，两者的话语体系、目的、读者都不同。

**学者向**：
- 话语：学术话语
- 目的：建立认知、解释因果
- 重点：讲道理、学理、哲理
- 读者：研究者、学者
- 特点：逻辑严密、论证充分、上升到理论层面

**领导向**：
- 话语：政治话语
- 目的：指导行动、统一思想
- 重点：谈认识、要求、举措
- 读者：领导干部
- 特点：简洁有力、操作性强、指向明确

动笔之前先明确方向，全程用同一套话语体系，不能混用。

**A1（past case）**
主题：关于"高质量发展"

**学者向写法**：
```
高质量发展是经济发展规律的历史性跃迁。

从经济学视角看，高质量发展意味着：
1. 效率提升：从要素驱动转向创新驱动
2. 结构优化：从增量扩张转向存量调整
3. 成果共享：从非均衡发展转向包容性发展

这一转变的学理基础在于马克思主义的经济发展理论...
```

**领导向写法**：
```
高质量发展是当前和今后一个时期的根本要求。

一、深刻认识高质量发展的重大意义（统一思想）
二、准确把握高质量发展的实践要求（明确任务）
三、切实强化高质量发展的工作举措（指出路径）
```

对比：学者向讲道理学理哲理，领导向谈认识要求举措。

**A2（future trigger）**
你要写一篇关于"基层治理现代化"的文章，需要根据不同对象选择不同方向：
1. 学术期刊发表
2. 领导讲话稿

请分别说明两个方向的写作要点。

**E（Execution）— 执行步骤**
1. **定方向**：根据文章目的和读者确定是学者向还是领导向
2. **选话语**：学者向用学术话语，领导向用政治话语
3. **定结构**：学者向按逻辑（从普遍到特殊），领导向按任务（认识→要求→举措）
4. **守一致**：全程用同一套话语体系，不混用

**B（Boundary）— 边界与局限**
- **失效场景**：日常沟通、简报等不需要这么严格的文体区分
- **常见错误**：学者向文章突然转向谈要求、领导向文章大篇幅讲学理
- **作者盲点**：认为"道理"是通用的，忽视了读者身份对话语体系的要求
