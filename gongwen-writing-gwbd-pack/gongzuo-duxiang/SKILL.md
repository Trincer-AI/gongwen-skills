# 工作分类矩阵法

**R（Reading）— 原文引用**
> 两个维度审视工作：成长性/重要性（上级关注度、考核权重）和个人的掌控力/完成度（熟悉程度、专业匹配度）

**I（Interpretation）— 理解重写**
工作分类矩阵用两个维度将工作划分为四类，帮你梳理出亮点、基点、潜力和痛点：

维度一：成长性/重要性（上级关注度、考核权重、对全局工作的影响力）
维度二：掌控力/完成度（个人对工作的熟悉程度、专业匹配度、独立完成能力）

四象限：
1. 高成长性×高掌控度=**代表作**（主动承担做出亮点的核心工作）→重点突出、浓墨重彩
2. 低成长性×高掌控度=**压舱石**（熟练琐碎但必不可少的基础事物）→合并归纳、提炼价值、强调稳准笑
3. 高成长性×低掌控度=**潜力股**（领导交办的不熟悉领域任务）→展示学习姿态、强调参与和探索
4. 低成长性×低掌控度=**成本项**（流程繁琐价值感低的内耗型工作）→强调转化优化、指向改进措施

**A1（past case）**
刚入职办公室新人的工作梳理：
- 【代表作】用excel做通讯录更新模板 → "坚持效能导向，在创新工作方法上实现突破"
- 【压舱石】收发文件、接听电话、整理档案 → "强化规范执行，在保障日常运转中筑牢基础"
- 【潜力股】协助筹备大会、写政务信息 → "注重实践锻炼，在参与重点任务中积累经验"
- 【成本项】填统计报表 → "立足岗位思考，在加强业务学习中提升质效"

**A2（future trigger）**
你是单位的中层干部，年终要写述职报告，需要用工作分类矩阵梳理亮点。

**E（Execution）— 执行步骤**
1. **列出所有工作**：不限岗位，把近期做的工作都列出来
2. **评估两个维度**：成长性（上级关注度、考核权重）和掌控度（熟悉程度、专业匹配度）
3. **划入四象限**：判断每项工作属于哪个象限
4. **按象限特点写作**：代表作重点突出、压舱石合并提炼、潜力股展示探索、成本项指向改进

**B（Boundary）— 边界与局限**
- **失效场景**：工作太简单的基层岗位
- **常见错误**：把所有工作都往"代表作"塞
- **作者盲点**：压舱石类工作虽然平常，但写好了同样有亮点
