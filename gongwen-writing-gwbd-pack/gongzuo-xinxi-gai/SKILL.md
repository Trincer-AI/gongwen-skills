# 工作信息改写法

**R（Reading）— 原文引用**
> 工作信息要突出"快准量简"，工作总结则要"清权理息正稳"。

**I（Interpretation）— 理解重写**
工作总结→工作信息的改写核心是思维转换：

| 维度 | 工作总结 | 工作信息 |
|------|----------|----------|
| 目的 | 全面系统回顾 | 快速决策支持 |
| 思维 | 清权理息正稳 | 快准量简 |
| 主体 | 事件/工作 | 责任主体明确 |
| 动词 | 实施（已完成） | 探索（创新突破） |
| 成效 | 过程性描述+概括性成效 | 结果性描述+具象化成效 |

**改写三步**：
1. 总起句：主体明确化 + 动作优化（"实施"→"探索新路径"）+ 成效优化（"赋能提质增效"→"激发新活力"）
2. 小标题：识别融合领域 + 提炼高价值成效词（如"释放新潜力/培塑新业态/打造新引擎"）
3. 具体内容：去支流干，突出行动名称和数据

**A1（past case）**
"创新实施文化加融合策略，赋能区域文旅消费提质增效" → "大观园区探索文化加新路径，激发文旅消费新活力"

**A2（future trigger）**
你有一篇企业服务工作总结，需要快速改写成工作信息上报。

**E（Execution）— 执行步骤**
1. **判断目的**：这份材料是给领导决策用（工作信息）还是全面回顾用（工作总结）
2. **总起句改写**：补齐责任主体 + 动词从"实施"改为"探索/创新" + 成效从概括改为具象
3. **小标题改写**：识别核心融合领域 + 每领域配一个高价值成效词
4. **内容改写**：去分析、去因果链，只保留行动名称和结果数据

**B（Boundary）— 边界与局限**
- **失效场景**：不是所有工作总结都能改写成信息，如果工作没有决策价值就不适合
- **常见错误**：只压缩不改思维，导致信息还是太像总结
- **作者盲点**：信息追求快准量简，可能牺牲完整性；有些场合需要完整的总结而非精简的信息
