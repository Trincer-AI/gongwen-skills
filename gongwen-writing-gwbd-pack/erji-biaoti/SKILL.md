# 二级标题六种逻辑维度法

**R（Reading）— 原文引用**
> 工作总结二级标题有六种逻辑维度：时序、事项、主体、区域、对象、成效。

**I（Interpretation）— 理解重写**
工作总结的二级标题有六种逻辑维度，可以从不同角度拆解工作内容。

核心原则：一文一维，同一层级只用一种维度，维度要统一。

六种逻辑维度：

1. **时序维度** — 按时间阶段划分（筹备期/实施期/验收期），体现工作推进的节奏
2. **事项维度** — 按具体工作划分（党建/业务/队伍），体现工作的具体内容
3. **主体维度** — 按责任主体划分（牵头部门/配合部门），体现责任分工
4. **区域维度** — 按空间划分（城区/乡镇/园区），体现空间布局
5. **对象维度** — 按服务对象划分（企业/群众/部门），体现服务指向
6. **成效维度** — 按工作成果划分（制度建设/能力提升/群众满意度），体现工作成果

维度选择建议：
- 时间节点清晰 → 用时序维度
- 多项工作并行 → 用事项维度
- 多主体参与 → 用主体维度
- 区域差异明显 → 用区域维度
- 服务对象多元 → 用对象维度
- 成果导向突出 → 用成效维度

**A1（past case）**
某县乡村振兴工作总结：

**错误维度选择**：
```
一、产业振兴（事项维度）
  1. 一季度做了什么（时序维度混用）
  2. 二季度做了什么
二、生态振兴（事项维度）
  1. 龙头企业带动（主体维度混用）
  2. 农民参与
```

问题：二级标题混用了时序维度和主体维度

**正确维度选择**：
```
一、产业振兴（成效维度）
  1. 产业规模化水平提升
  2. 农业龙头企业培育
  3. 农产品品牌打造
二、生态振兴（成效维度）
  1. 人居环境整治
  2. 生态保护修复
  3. 绿色发展模式
```

统一用成效维度，每个二级标题都是"某方面取得某成效"的结构。

**A2（future trigger）**
某单位"优化营商环境"工作总结，涉及：
- 制度建设（出台了一系列政策）
- 流程再造（压缩了办事时限）
- 队伍建设（组织了业务培训）
- 信息化建设（上了政务平台）
- 监督检查（开展了专项检查）

请选择最合适的维度并设计二级标题体系。

**E（Execution）— 执行步骤**
1. **分析工作特点**：看哪方面特点最突出（时间/事项/主体/区域/对象/成效）
2. **选择维度**：从六种维度中选择最匹配的一种
3. **统一层级**：确保同一层级只用一种维度
4. **拟写标题**：标题风格统一，体现逻辑关系

**B（Boundary）— 边界与局限**
- **失效场景**：综合性报告需要多维度时（此时应分板块而非统一维度）
- **常见错误**：同一篇总结里混用多种维度、维度选择与工作特点不匹配
- **作者盲点**：追求标题的工整对仗而忽视了维度的统一性
