# 亮点深化三模型

**R（Reading）— 原文引用**
> 把工作亮点写得足够亮，有三个模型：辩证呈现（变不利为有利）、系统集成（单点突破→系统性协同）、时空纵深（现状+未来+生态）。

**I（Interpretation）— 理解重写**
三种亮点深化模型：

**模型一：辩证呈现（约束转化）**
- 核心：变不利为有利，化约束为勋章
- 方法：主动承认约束条件，作为背景反向烘托成绩的非凡性
- 句式：框架A困境突破式（在X约束下实现了Y成果）/ 框架B层级递进式（不仅...更...）

**模型二：系统集成（多维协同）**
- 核心：将单点成绩整合为系统性框架
- 方法：要素拆解 + 关系定义（动词+品牌概念）+ 体系整合
- 品牌概念三原则：功能性、价值性、形态化

**模型三：时空纵深（三层结构）**
- 核心：构建有层次的成果体系
- 方法：基本盘（现状厚度）+ 增长极（未来高度）+ 生态圈（系统活力）
- 五步骤：原料库→基本盘→增长极→生态圈→逻辑串联

**A1（past case）**
辩证呈现："在减量约束下实现提质，在财政压力下跑出加速度"
系统集成：以盐城融合建设策源地，以央城融合牵手国家队...
时空纵深：一是三大千亿级主导产业聚集城市，二是未来产业前瞻领跑

**A2（future trigger）**
写一段关于"城市更新"的亮点，需要体现规模成效+创新突破+治理生态。

**E（Execution）— 执行步骤**
1. **判断用哪个模型**：
   - 有约束条件但取得成绩 → 辩证呈现
   - 有多个协同维度 → 系统集成
   - 要体现厚度+高度+活力 → 时空纵深
2. **辩证呈现**：识别约束条件 + 界定卓越成果 + 构建对立统一句式
3. **系统集成**：拆解关键维度 + 为每维度配动词+品牌 + 一级标题升华
4. **时空纵深**：准备原料库 + 构建基本盘/增长极/生态圈 + 逻辑串联

**B（Boundary）— 边界与局限**
- **失效场景**：没有约束条件时不用辩证呈现；工作单一维度不用系统集成；简单工作不用时空纵深
- **常见错误**：三个模型混用或叠加；用系统集成但各维度没有协同关系
- **作者盲点**：模型是理论归纳，实际可能需要组合使用
