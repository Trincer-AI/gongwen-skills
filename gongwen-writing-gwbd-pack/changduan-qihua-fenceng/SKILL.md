# 长短期谋划分层法

**R（Reading）— 原文引用**
> 5年谋划聚焦战略层（做什么/为什么做），明年谋划聚焦执行层（怎么做/何时做/做到什么程度）

**I（Interpretation）— 理解重写**
在一篇材料中兼顾长短期谋划有四个难点及解决方法：

**难点一：内容衔接与避免重叠**
- 5年谋划提原则 → 明年谋划抓落实
- 5年谋划列项目 → 明年谋划推进度
- 5年谋划定范畴 → 明年谋划定重点

**难点二：详略尺度精准把控**
- 5年谋划：虚中有实（框架实，细节活）— 为未来留弹性
- 明年谋划：实中有虚（执行实，宏观略）— 防止迷失于琐碎

**难点三：逻辑架构并行与区分**
- 5年谋划：按领域/体系/功能维度铺开（全面系统的宏观框架）
- 明年谋划：按工作模块/行动线索/优先级排序（聚焦重点的微观关节）

**难点四：表述风格切换与统一**
- 5年谋划：战略性/前瞻性/概括性/原则性（描绘蓝图）
- 明年谋划：执行性/务实性/细节性/措施性（指令动作）

**A1（past case）**
林下经济发展：
- 5年谋划（提原则）："坚持生态筑基、产业增效，大力发展以林下种植、林下养殖为代表的林下立体种养模式"
- 明年谋划（抓落实）："扩面建设五个林下药材示范基地，种植黄精茯苓等中药材2000亩；建成30个标准化林下菌大棚"

**A2（future trigger）**
你要写一份经济发展五年规划加明年计划，需要在一篇材料中体现长短兼顾。

**E（Execution）— 执行步骤**
1. **内容分层**：5年谋划定义方向和边界，明年谋划明确动作和节点
2. **详略把控**：5年虚中有实（留弹性），明年实中有虚（防迷失）
3. **架构区分**：5年按战略维度铺开，明年按执行模块聚焦
4. **风格切换**：5年用战略性语态，明年用执行性语态

**B（Boundary）— 边界与局限**
- **失效场景**：如果材料只需要短期谋划，不必强行加入长期视角
- **常见错误**：两者内容重叠（没有分层）、详略失衡（要么太虚要么太实）
- **作者盲点**：战略层与执行层的平衡需要根据具体场合灵活调整
