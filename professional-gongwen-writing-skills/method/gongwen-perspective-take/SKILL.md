# 换位技巧

## R — 原文

"向上带入，就是为哪个领导写稿，就把自己假想成哪个领导。这个假想，不是假装想象，而是把桌面收拾整洁，倒一杯水，坐在办公桌前，端正坐姿——此时此刻，你就是领导本人。"

## I — 方法论骨架

**换位技巧两层次** = 向上代入（领导视角）+ 向下代入（执行者视角）

向上代入：把自己当成领导，思考他会怎么讲、怎么要求

向下代入：把自己当成执行者，问自己能不能听懂、能不能落实

关键原则：宁可没有站位，不能站错位（站错位=给其他部门布置工作）

## A1 — 书中的应用

向上代入操作：收拾桌面、倒杯水、端正坐姿——想象自己就是领导本人

站位五层次援引：党中央→系统最高机关→省级→地市级→本单位（基层引用2-3个层次）

两种典型错误：
- 写了下属的话（让下属无话可说）
- 写了过细的话（显得领导没水平没思路）

## A2 — 触发场景

**秘书场景**："领导让我写这份材料，但我不确定他想要什么高度"

**思维顾问场景**："如何快速理解领导意图？站位高低怎么判断？"

**角色扮演场景**："假设你是领导，站在你的角度批注这份材料"

## E — 可执行步骤

### Step 0 - 预检

**Input**: 待写材料
**Output**: 问题诊断（是否需要换位 + 换哪个位）

- [ ] "材料被领导批评过'没有高度'吗？"是→需要换位
- [ ] "完全理解领导意图吗？"否→需要换位
- [ ] "是终稿阶段吗？"是→无法大幅修改，换位价值低

**质量检查点**：
- [ ] "这个问题是定位不清还是高度不够？"定位不清→先回到 fw-001 结构法
- [ ] "还有时间大幅修改吗？"无→只做局部调整

### Step 1 - 向上代入（领导视角）

**Input**: 材料 + 领导信息
**Output**: 领导视角分析表（会怎么讲 + 希望传递什么）

**操作仪式**：收拾桌面 → 倒杯水 → 端正坐姿 → 假想自己是该领导

- [ ] "该领导的历史讲话看过吗？"（目标≥2篇，分析语言风格）
- [ ] "该领导本次想传递的核心信息是什么？"（目标≥1个关键词）
- [ ] "该领导会怎么开场？"（参考帽段写法）
- [ ] "该领导会强调什么？"（参考重点词频率）

**质量检查点**：
- [ ] "自己的语言风格和领导风格一致吗？"否→改用领导的常用词
- [ ] "材料站位是否在领导权限范围内？"否→降级或改用上级表述

### Step 2 - 站位检查

**Input**: 领导视角分析表
**Output**: 站位风险点清单

**站位五层次**（从上到下引用2-3个层次）：
- 党中央 → 系统最高机关 → 省级 → 地市级 → 本单位

- [ ] "援引的站位层次在领导权限范围内吗？"（地市级领导不能引用党中央具体要求）
- [ ] "是否写了'让下属无话可说'的话？"是→删除或降级
- [ ] "是否写了'过细的话'（显得领导没水平）？"是→删除或泛化
- [ ] "宁可没有站位，不能站错位"——这句话记住了吗？

**质量检查点**：
- [ ] "这段话是给领导增彩还是给领导挖坑？"增彩→保留；挖坑→删除
- [ ] "执行者看到这段会怎么想？"（如果觉得"说得比我领导还好"→站错位）
- [ ] "这段话是否超越了本级权限？"是→删除

### Step 3 - 向下验证（执行者视角）

**Input**: 调整后材料
**Output**: 可执行性评估

- [ ] "执行者能看懂吗？"（如果看不懂→改写为直白表述）
- [ ] "执行者能落实吗？"（如果落实不了→补充具体操作说明）
- [ ] "有没有歧义表述？"（如果有多人可以理解→改写为唯一理解）

**质量检查点**：
- [ ] "执行者看完后第一反应是'懂了'还是'这啥意思'？"这啥意思→改
- [ ] "执行者需要额外问领导才能落实吗？"是→补充细节
- [ ] "每项要求都有明确的责任主体吗？"否→补充

### Step 4 - 调整完善

**Input**: 站位检查 + 可执行性评估结果
**Output**: 最终稿

- [ ] "用领导的话写领导的事了吗？"（避免用自己的话替领导说）
- [ ] "扩句、补句、拔高都在领导既定框架内了吗？"（不超过权限）
- [ ] "语言风格和该领导历史讲话一致了吗？"（对照检查）
- [ ] "终稿读一遍，假设你是执行者，能落实吗？"否→回退 Step 3 修改

## B — 边界与Fallback

**边界**（场景|触发条件|处理动作）：

| 场景 | 触发条件 | 处理动作 |
|---|---|---|
| 不了解领导风格 | 没有看过该领导历史讲话 | 查阅该领导历史讲话稿（目标≥2篇） |
| 站位拿不准 | 不确定这句话是否越权 | 改用上级原文表述，使用"上级的话写上级的事" |
| 担心站错位 | 重要材料（如汇报上级） | 先与领导确认大方向再动笔 |
| 材料已终稿 | 修改空间极小 | 只做局部词语调整，不做大改动 |
| 时间极紧 | 紧急材料 | 直接套用该领导历史讲话结构 |
| 超越权限 | 写了让其他部门执行的内容 | 立即删除，改写为配合而非布置 |
| 站位过度 | 拔高超过领导权限 | 降级到本级能表述的层次 |

**Fallback**：
- 不了解该领导风格 → 查阅该领导历史讲话稿
- 站位拿不准 → 查阅上级文件，使用"上级的话写上级的事"
- 担心站错位 → 先与领导确认大方向再动笔
