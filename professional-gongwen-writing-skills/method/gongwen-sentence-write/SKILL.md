# 句子撰写方法

## R — 原文

"给句子做减法，就是逐句反查，不断地问自己：'这个内容可不可以删除？'如果删除后，句子依然通顺、意思表述依然清晰，果断删！"

"语言要平实——能要部分数据，不要整篇公文。"

## I — 方法论骨架

**减法写作原则** = 逐句反查 + 能删则删 + 意思清晰

句子结构原则：头轻脚重（短句+长句，而非长句+短句）

语言风格：平实得体，避免假大空，用精准表述替代模糊表述。

## A1 — 书中的应用

减法核心心态：敢于"壮士断腕"，追求"多一分则肥，少一分则瘦"的公文境界。

出声音朗读是检验句子的有效方法——能发现屏幕浏览时忽略的拗口处。

注意句子结构：先短后长，避免头重脚轻。

## A2 — 触发场景

**秘书场景**："这句话读起来怪怪的，帮我改顺"

**思维顾问场景**："公文句子和普通文章句子的区别是什么？怎么写得规范得体？"

**角色扮演场景**："假设你是领导，朗读这段话并指出拗口之处"

## E — 可执行步骤

### Step 0 - 预检

**检查点**: 用户是否处于以下状态？
- [ ] 句子拗口、不通顺
- [ ] 句子过长（一行以上）
- [ ] 口语化/不规范的表述

**若全部未勾选**: 切换到 `gongwen-proofread`（可能需要整体复核）

---

### Step 1 - 减法检查（核心：能删则删）

**输入**: 待检查的完整句子
**输出**: 精简后的句子，标注删除部分

**操作步骤**：
1. 逐句朗读，问自己："这句话哪个词/短语删掉后，句意不变？"
2. 对每个疑似冗余词执行"删除测试"——删掉 → 读一遍 → 意思不变 → 删
3. 检查句子结构：
   - ✅ 正确：主语+（状语）+ 谓语 + 宾语（短）+ 补充说明（长）
   - ❌ 错误：长长长长状语 + 主语 + 谓语（头重脚轻）

**质量检查点**：
- [ ] "删掉这个词/短语后，句意改变了吗？"没改变→删
- [ ] "这句话的主谓结构完整吗？"不完整→补
- [ ] "句子是否'头轻脚重'？"头重→调整语序

---

### Step 2 - 规范打磨（核心：精准替代）

**输入**: Step 1 精简后的句子
**输出**: 规范化句子

**操作步骤**：
1. 出声朗读（不是默读，是真的发出声音读），读到哪里卡住哪里就是问题点
2. 问题词汇分类：
   - 模糊动词 → 精准动词（"搞"→"开展" / "弄"→"处理"）
   - 重复用词 → 替换或合并
   - 口语化 → 书面语（"大概"/"可能"→"若干"/"部分"）
3. 标点检查：顿号/分号/逗号用法是否正确

**质量检查点**：
- [ ] "出声朗读时，有没有绊口/拗口的地方？"有→标记并修改
- [ ] "模糊动词是否都替换了？"未→继续
- [ ] "标点使用是否符合公文规范？"否→查标点规范

---

### Step 3 - 最终校验（核心：通读确认）

**输入**: Step 2 规范化后的完整段落
**输出**: 可交付的句子

**操作步骤**：
1. 完整朗读全段（不是单句，是串联上下文读）
2. 检查点：
   - 段落首句是否简洁有力？拖沓→回Step 1重检
   - 段落末句是否收得住？散了→补一句总结
3. 格式终查：数字用法（"一是"不是"第一"）/ 标点（全角/半角）

**质量检查点**：
- [ ] "全段通读后，语感流畅吗？"否→从头
- [ ] "每句话都能'一口气读完'吗？"不能→缩句
- [ ] "公文格式（数字/标点）是否规范？"否→查《党政机关公文格式》

## B — 边界与Fallback

| 场景 | 触发条件 | 处理动作 |
|---|---|---|
| 情感类文章 | 用户说"写演讲稿/致辞" | 切换到情感写作模式，句子可以更长、更优美 |
| 句子拗口但不知如何改 | 出声朗读后发现问题 | 让语感引导修改——"读着别扭的地方就改" |
| 不确定是否该删 | 句子似乎冗余但担心改变意思 | 暂时保留，通读全文后再判断 |
| 语言不够规范 | 口语化表述不知道如何改 | 参照本单位历史公文的相似表述 |
| 标点符号拿不准 | 顿号/分号/引号用法不确定 | 查《党政机关公文格式》国家标准 |

**核心边界**：
- 句子必须服务于内容，不能为精简而精简
- 核心观点句（结论句）不可删
- 数据/案例/名称不可删（这些是"实"的基础）
