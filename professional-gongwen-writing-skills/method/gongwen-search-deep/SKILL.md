# 深度检索法

## R — 原文

"原来，材料高手都有自己的'兵器库'。每年年底，负责公文写作工作的执笔人都压力如山...对于一些重要性相对较低的公文材料，比如新春慰问活动的讲话稿，可以通过检索的方式，找一些成型的模板，把本单位的工作融合进去。"

"真正有效的搜索是'微观搜索'，比如某两个标题对仗不够工整，需要换个词语，可以用已经确定的词语去搜索，找到最契合的同义词、近义词；又如某段内容需要写培训方式的亮点，可以用'培训经验'或'培训亮点'几个字进行搜索。"

## I — 方法论骨架

**素材搜索微观法** = 宏观搜索（找方向）+ 微观搜索（找精准素材）

素材来源八分类：上级（帽段参考）、领导（风格学习）、下级（数据复核）、同行（横向比较）、业务部门（补实）、职能部门（补虚）、内网（研读）、外网（备不时之需）。

搜索渠道三优先：site:gov.cn政府网站、微信政务号（"一张图"+关键词）、分段式搜索核心词。

核心心态：检索不是为了照搬照抄，而是认识事物的第一步。

## A1 — 书中的应用

作者建议每周末整理素材库，半年可充盈"兵器库"。按文种和结构分类整理，形成快速调取能力。对于新春慰问活动讲话稿等低重要性公文，直接检索成型模板融入本单位工作即可。

微观搜索案例：标题对仗不工整时，用已确定词语搜索同义词近义词；写培训亮点时，用"培训经验""培训亮点"精准搜索。

## A2 — 触发场景

**秘书场景**："领导要一份关于乡村振兴的工作报告，我从没写过这个主题，帮我找找参考"

**思维顾问场景**："如何高效积累公文素材库？有什么搜索渠道推荐？"

**角色扮演场景**："假设你是一名资深办文人员，演示如何用微观搜索找到精准素材"

## E — 可执行步骤

### Step 0 - 预检

**Input**: 写作任务
**Output**: 搜索策略选择（宏观/微观/分段）

- [ ] "搜索目的是找方向还是找精准表述？"方向→宏观搜索；精准→微观搜索
- [ ] "文种重要性：低重要性（如新春慰问）可直接套模板，高重要性需深挖"
- [ ] "本单位历史公文是否存在？"是→优先内网，跳过外网搜索
- [ ] "时间紧迫度？"极高→先搭框架再补充素材

**质量检查点**：
- [ ] "这个问题已经有成熟方案了吗？"是→直接用历史公文
- [ ] "这个问题属于创新性课题吗？"是→需要宏观搜索建立认知

### Step 1 - 选择搜索策略

**Input**: 搜索目的 + 文种重要性
**Output**: 搜索策略路径

| 搜索类型 | 触发条件 | 关键词特征 | 预期结果 |
|---|---|---|---|
| 宏观搜索 | 陌生领域、首次接触 | 大方向词（"党建文化""乡村治理"） | 建立认知、找到框架 |
| 微观搜索 | 细节打磨、对仗不工整 | 具体词（"培训经验""培训亮点"） | 精准素材、同义替换 |
| 分段搜索 | 每段核心内容 | 核心词单独搜索 | 海量素材支撑每句 |

- [ ] "选择了宏观搜索？→直接搜大方向词，目标找到3+篇参考"
- [ ] "选择了微观搜索？→用具体词语找精准素材，目标找到5+个可用例句"
- [ ] "选择了分段搜索？→每段核心词单独搜索，确保每句话都有海量素材"

**质量检查点**：
- [ ] "搜索结果>10条吗？"否→扩大关键词范围
- [ ] "找到的素材和本单位工作相关吗？"否→换关键词重新搜索

### Step 2 - 精准渠道锁定

**Input**: 搜索策略 + 关键词
**Output**: 筛选后的素材列表

**渠道优先级**：
1. 内网：本单位历史公文（最贴近、可直接用）
2. 政府网站：site:gov.cn + 关键词（权威、可参考结构）
3. 微信政务号：搜索"一张图"+关键词（时效性强、接地气）
4. 外网：备不时之需（无合适内网时）

- [ ] "site:gov.cn 搜索返回几条结果？"（目标≥3条）
- [ ] "微信政务号找到几张'一张图'？"（目标≥2张）
- [ ] "内网有历史公文吗？"有→优先使用，标记来源

**质量检查点**：
- [ ] "找到的素材是否偏离本单位实际？"偏离→换渠道
- [ ] "素材来源是否权威？"（政府网站>微信>外网）

### Step 3 - 素材入库

**Input**: 筛选后的素材
**Output**: 个人素材库（按文种+结构分类）

**分类维度**：
- 按文种：简报/总结/讲话/思路/通知...
- 按结构：帽段/意义/问题/对策/要求/结尾

- [ ] "素材已按文种分类？"（每个文种≥3篇）
- [ ] "素材已按结构分类？"（每个结构≥2段）
- [ ] "每周末整理一次？"（确保素材库持续更新）

**质量检查点**：
- [ ] "入库素材能否在30秒内调取？"否→重新分类或标注关键词
- [ ] "素材库已有多少条？"（目标：半年后≥100条）

**量化目标**：
- 每周积累：≥5条新素材
- 半年目标：≥100条可用素材
- 调用速度：≤30秒定位目标素材

### Step 4 - 素材复用与迭代（可选步骤）

**触发条件**：当搜索结果不理想或需要更高质量素材时调用

**Input**: 低质量搜索结果 或 需迭代优化的素材
**Output**: 高质量可用素材 + 迭代策略

**操作步骤**：
1. 分析失败原因：关键词太宽泛/太窄？渠道选择不当？
2. 迭代搜索：
   - 宽泛→缩小：增加限定词（"2024年"+关键词+"工作要点"）
   - 狭窄→扩大：去掉限制词或换同义词
3. 多渠道交叉验证：政府网站+微信+内网三方印证
4. 素材质量评级：
   - A级：政府网站+内容匹配+可直接用
   - B级：需修改后可用（替换单位名称等）
   - C级：仅供参考（结构/思路）

**质量检查点**：
- [ ] "迭代后搜索结果>5条？"否→继续调整关键词
- [ ] "找到的素材能否交叉验证？"能→置信度提高
- [ ] "素材质量评级≥B级？"否→考虑降低使用优先级

**Fallback**：若迭代3次仍无合适素材，切换到"套用模板+现场补充"模式

## B — 边界与Fallback

**边界**（场景|触发条件|处理动作）：

| 场景 | 触发条件 | 处理动作 |
|---|---|---|
| 找不到合适素材 | 搜索结果<3条 | 回到 fw-003 兵器库查找类似文种模板 |
| 时间紧迫 | 紧急材料 | 先搭框架（目标文种结构），再补充素材 |
| 完全陌生领域 | 首次接触该主题 | 从上级文件中找"帽段"参考 |
| 搜索偏离主题 | 结果与本单位工作无关 | 换关键词或换渠道 |
| 素材质量差 | 找到的素材不权威 | 优先使用政府网站和内网，外网素材需核实 |
