# 问题导向式思维法

**R（Reading）— 原文引用**
> 问题导向式思维即不回避问题，将问题视为机遇突破口和出发点，通过直面问题、剖析问题来推动工作进展，实现创新发展。

**I（Interpretation）— 理解重写**
问题导向式思维是一种**转化视角**的写作方法：

**核心公式**：[问题名词] 是 [工作领域] 的 [着力点/切入点/落脚点/攻坚点]

| 问题类型 | 转化示例 | 着力点类型 |
|----------|----------|------------|
| 落后产能的瓶颈 | 产业升级的攻坚点 | 攻坚点 |
| 企业经营困境 | 营商环境优化的着眼点 | 着眼点 |
| 老旧小区破旧面貌 | 城市更新的切入点 | 切入点 |
| 交通拥堵节点 | 城市治理精细化的关键点 | 关键点 |
| 弱势群体生活难题 | 民生兜底保障的聚焦点 | 聚焦点 |

**作用**：把负面表述转化为积极导向，凸显责任担当，拉近政府与群众的距离。

**A1（past case）**
短板就是发力点，消费者之忧也是产业创新的空间。

落后产能的淘汰瓶颈是产业升级的攻坚点。

**A2（future trigger）**
写一个关于"如何解决停车难问题"的观点句。

**E（Execution）— 执行步骤**
1. **识别问题**：找到工作中存在的问题或短板
2. **选择着力点类型**：攻坚点（最难的地方）/ 切入点（最容易突破的地方）/ 着眼点（关注的核心）/ 落脚点（最终目的）/ 关键点（最关键的地方）/ 聚焦点（最需要关注的人群）
3. **套用公式**：将问题与工作领域用"是...的..."连接
4. **检查逻辑**：转化要有逻辑基础，问题确实能带来机遇时才转化，不能强行转化

**B（Boundary）— 边界与局限**
- **失效场景**：没有问题或劣势的工作不需要强行找问题；纯粹的成绩展示不需要转化
- **常见错误**：强行转化（污染严重说是生态保护的机遇，听起来牵强）；没有问题回避问题（本来没有问题的领域硬要找问题）
- **作者盲点**：转化思维不是万能的，有些问题就是单纯的困难，无法转化；过度使用可能让文章显得套路化
