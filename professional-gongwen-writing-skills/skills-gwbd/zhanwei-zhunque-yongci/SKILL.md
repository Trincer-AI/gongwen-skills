# 站位准确用词法

**R（Reading）— 原文引用**
> 在构词的两端，前端从贯彻到落实到按照根据结合等一系列介词是层层降维，末端从指导思想到精神到要求到部署到安排，也是层层降维。在同一语境中，不同的层级、不同的站位用词要准确。

**I（Interpretation）— 理解重写**
站位准确是公文写作的第一要务。用词必须与作者/领导的层级相匹配，错误用词会给领导"挖坑"。

词汇降维体系：
- **前端介词**：贯彻 → 落实 → 按照 → 根据 → 结合（层级递减）
- **末端名词**：指导思想 → 精神 → 要求 → 部署 → 安排（层级递减）

使用时，高层级用靠前的词，低层级用靠后的词，不可混用。

**A1（过去案例）**
写关于"落实市委书记重要讲话精神"的内容。错误：把"重要讲话精神"用于市委书记（这是给领导挖坑）。正确：应使用"工作要求""工作部署"等词汇。

**A2（未来触发）**
写一份关于"传达学习省委书记指示精神"的通知，需要选择合适的介词和名词搭配。

**E（Execution）— 执行步骤**

| 步骤 | 操作 | 检查点 |
|------|------|--------|
| 1 | **确定站位**：写这份公文代表哪个层级？（中央/省级/市县级） | 站位要准确，不能"戴高帽"或"降格" |
| 2 | **选择介词**：高层级用"贯彻"，低层级用"落实""按照""根据" | 介词要与站位匹配，不可混用 |
| 3 | **选择名词**：高层级用"精神"，低层级用"要求""部署""安排" | 名词要与站位匹配，不可越级 |
| 4 | **检验匹配**：代入语境检查是否合适，有无"给领导挖坑" | 检验标准：代入后读起来是否自然合理 |

**词汇降维参考**：
- 前端介词（高层→低层）：贯彻 → 落实 → 按照 → 根据 → 结合
- 末端名词（高层→低层）：指导思想 → 精神 → 要求 → 部署 → 安排

**B（Boundary）— 边界与局限**
- **失效场景**：纯事务性通知（不涉及领导表态）不需要考虑站位
- **常见错误**：将"重要讲话精神"用于非最高领导；将"全面部署"用于副职会议
- **作者盲点**：过度强调站位可能让文章显得程式化，在需要创新表达的场合可能不合适
