# 重点工作谋划四方面法

**R（Reading）— 原文引用**
> 谋划重点工作不是要破我们自己的题，是要想如何破领导心中的题。领导心中的题离不开如何服务和融入国家战略。

**I（Interpretation）— 理解重写**
重点工作谋划有四个方面：

| 方面 | 核心内容 | 操作要点 |
|------|----------|----------|
| 上级考核 | 对照考核标准，分析差距，识别重点领域 | 研读考核文件，明确指标权重，对照差距 |
| 落实要求 | 从上级讲话稿、会议纪要中找落实角度 | 研究对口部门讲话，找落实切入点 |
| 试点示范 | 争取试点示范项目和通报表彰 | 提前了解申报条件，结合本地优势准备 |
| 对接国家战略 | 具体业务承载+共性问题破解 | 找到与本地职能业务的契合点 |

核心原则：不是破自己的题，是破领导心中的题。

**A1（past case）**
某市在谋划重点工作时的四方面思考：
- 上级考核：对照省里考核指标，识别营商环境是短板
- 落实要求：研究省里领导讲话，"放管服"改革是重点
- 试点示范：争取省级营商环境试点示范
- 对接国家战略：以"数字政务"为切入点服务全国统一大市场建设

**A2（future trigger）**
为某县谋划下一年度重点工作，需要从四个方面思考并找到契合点。

**E（Execution）— 执行步骤**
1. **研读考核**：详细研读上级考核文件，明确指标要求
2. **研究讲话**：找上级对口部门讲话稿、会议纪要，找落实角度
3. **对接试点**：与上级业务部门沟通，了解试点项目申报条件
4. **连接战略**：思考本地工作如何服务和融入国家战略
5. **形成重点**：综合四方面，确定2-3个重点工作方向

**B（Boundary）— 边界与局限**
- **失效场景**：日常工作不需要这么系统地谋划；突发应急工作需要临时处理
- **常见错误**：只从自己业务出发，不考虑上级考核和领导关心的问题
- **作者盲点**：过度强调"破领导心中的题"可能导致迎合领导，失去独立判断
