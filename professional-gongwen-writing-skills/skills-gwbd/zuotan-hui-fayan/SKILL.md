# 座谈会发言四方法

**R（Reading）— 原文引用**
> 价值内核驱动法让不会写的人有故事可以讲，价值践行具象法让不会写的人有框架可以填。关键是根据出席座谈会的主要领导的审美风格与关注焦点选择合适的方法。

**I（Interpretation）— 理解重写**
四方法及适用领导类型：

| 方法 | 核心特点 | 适用领导 | 关键词 |
|------|----------|----------|--------|
| 价值内核驱动法 | 根源性叙事，精神传承 | 理论型/情怀型 | 根与魂/精神源头/个体成长 |
| 矛盾辩证分析法 | 对立统一破解困境 | 实干型/专家型 | 危与柱/低与高/破与立 |
| 能力智效提升法 | 量化指标，效能提升 | 业绩导向型 | 学以致用/求真务实/廉洁高效 |
| 价值践行具象法 | 品格坐标，行为指南 | 感召型/激发型 | 责任心/进取心/敬畏心 |

**核心选择逻辑**：领导的审美风格 + 你擅长的思维方式。

**A1（past case）**
价值内核驱动法："担当的根在于对党忠诚和对事业的责任感" + 个人成长故事（精神传承）

矛盾辩证分析法：书斋空心化 vs 经验碎片化 → 打通学用转化关节（辩证统一）

能力智效提升法："学生一点，言透一题，用活一招" + "一线蹲点查实情，问题导向谋实策"

价值践行具象法："践行担当要做到三心：责任心（恪尽职守）+ 进取心（不甘平庸）+ 敬畏心（守红线）"

**A2（future trigger）**
你将参加一场年轻干部座谈会，出席的领导重视"精神传承和理论深度"。你擅长讲故事。

**E（Execution）— 执行步骤**
1. **判断领导类型**：
   - 重视历史传承/理论深度/精神感召 → 理论型/情怀型
   - 注重实际问题/解决成效/逻辑思辨 → 实干型/专家型
   - 关注工作实效/能力突破/结果导向 → 业绩导向型
   - 强调精神面貌/价值认同/内生动力 → 感召型/激发型
2. **选择发言方法**：
   - 理论型/情怀型 → 价值内核驱动法（根源性叙事）
   - 实干型/专家型 → 矛盾辩证分析法（问题解决）
   - 业绩导向型 → 能力智效提升法（量化指标）
   - 感召型/激发型 → 价值践行具象法（品格坐标）
3. **结合个人优势**：会讲故事 → 价值内核；善用辩证 → 矛盾辩证；擅长数据 → 能力智效；长于框架 → 价值践行具象
4. **按四步骤执行**（以价值内核驱动法为例）：紧扣主题提取精神核心→深刻阐述认识→具体实践印证→持续深化表态

**B（Boundary）— 边界与局限**
- **失效场景**：不是座谈会的一般性工作汇报不需要此方法；文学创作不适用
- **常见错误**：方法与领导类型不匹配（给理论型领导用矛盾辩证法，可能被认为太实缺温度）
- **作者盲点**：方法选择有主观性，实际领导可能比分类更复杂，不能生搬硬套
