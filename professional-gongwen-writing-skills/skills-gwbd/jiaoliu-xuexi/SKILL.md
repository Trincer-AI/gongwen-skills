# 交流研讨学习思维法

**R（Reading）— 原文引用**
> 写交流研讨材料：把材料还原为领导消化吸收新知识、联系工作实际、形成行动纲领的认知过程

**I（Interpretation）— 理解重写**
交流研讨材料的本质是学了新知识之后怎么干的的思想汇报。

学习思维四维度：
1. **新知识点如何为你们赋能**：涵盖教育政策制定/教育体系改革/资源分配/质量评估等维度，让新知识为原本职能作用发挥得更好
2. **新知识点如何破解老问题**：为老难题提供新的解题思路，从扭转工作导向上存在的偏差入手
3. **新旧知识点如何关联**：与长期坚持的基础理论叠加激活，产生新的工作切口
4. **拆解新知识点本身**：从为谁干/干成什么样/靠什么干的逻辑路径拆解

**A1（past case）**
住建部门学习新发展理念的交流发言框架：
- 以新理念为定盘之星，在托举全面成长上展现新作为
- 以新理念为民心之境，在优化资源配置上取得新成效
- 以新理念为度量之衡，在赋能城市发展上实现新突破

**A2（future trigger）**
你要为科技局领导写一份学习"科技创新重要论述"的交流研讨材料。

**E（Execution）— 执行步骤**
1. **赋能视角**：新知识如何为科技工作赋能（政策制定/体系改革/资源配置/质量评估）
2. **破解视角**：新知识如何破解科技部门的老问题（研发投入强度上不去/成果转化打不通）
3. **关联视角**：新旧知识如何叠加激活（与基础理论关联产生新切口）
4. **拆解视角**：从为谁干/干成什么样/靠什么干拆解
5. **拟标题**：用精炼对仗的短语拟并列标题

**B（Boundary）— 边界与局限**
- **失效场景**：简单的工作布置不需要这么重的学习思维
- **常见错误**：四个维度面面俱到但没有重点
- **作者盲点**：学习思维的核心是"学以致用"，不是"学而空谈"
