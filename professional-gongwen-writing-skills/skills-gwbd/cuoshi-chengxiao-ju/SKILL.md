# 举措成效句边界法

**R（Reading）— 原文引用**
> 举措成效句要锚定举措对系统边界的影响，写成效句：定系统→划边界→问成效。

**I（Interpretation）— 理解重写**
举措成效句是描述工作举措带来的成效的句子。写好成效句的关键是**边界聚焦**——不越界承诺外部系统性影响。

三步法：
1. **定系统** — 明确工作的核心系统是什么，核心功能是什么
2. **划边界** — 系统的边界在哪里（服务供给与群体状态之间的作用面 / 资源投入与个体资本之间的转化界面 / 资金流转与需求满足之间的保障界面）
3. **问成效** — 基于具体举措和数据，追问让边界本身发生了何种可观测可描述的积极变化

**边界聚焦原则**：
- 可以写：边界面上呈现的活力状态、边界上完成的资源到资本的转化结果、系统自身核心功能在边界上的效能显现、举措对边界支撑平台的加固
- 不能写：促进社会公平正义、提升民生幸福感、增强城市吸引力等外部系统性影响

**A1（past case）**
**错误写法（越界）**：
"通过出台人才政策，提升了城市吸引力和竞争力，促进了社会公平正义"

分析：城市吸引力、社会公平正义是外部系统性影响，超出了举措的直接作用边界。

**正确写法（边界聚焦）**：
"通过出台人才政策，建成市级人才服务站点12个，实现高层次人才认定周期缩短40%，人才服务事项'一网通办'覆盖率达95%"

分析：描述的是举措对边界（人才服务体系）的直接加固，是可观测的、可验证的成效。

**A2（future trigger）**
某单位实施了"社区养老服务提升"工程，涉及：
- 建成社区养老服务中心15个
- 培训养老护理员200名
- 老年人满意度达92%

请用边界聚焦原则，写3条举措成效句。

**E（Execution）— 执行步骤**
1. **定系统**：明确这是什么系统（如养老服务体系、人才服务系统）
2. **划边界**：找到系统的作用面（如服务供给与老年人需求之间的界面）
3. **问成效**：具体举措+数据，追问边界上的可观测变化
4. **查越界**：确认没有描述外部系统性影响（幸福感、吸引力等）

**B（Boundary）— 边界与局限**
- **失效场景**：宏观总结、战略规划类文本不需要这么严格的边界控制
- **常见错误**：越界去写"提升了群众幸福感"、"增强了城市影响力"等模糊的外部效应
- **作者盲点**：认为写大一点的效果显得更有成绩，实际上领导更认可具体可核查的边界内成效
