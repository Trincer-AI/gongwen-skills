# 动宾集群写作法

**R（Reading）— 原文引用**
> 公文写作的顶级思维：将工作中涉及的一切可支配可影响的要素都视为我这个能动主体可以主动施加动作的客体

**I（Interpretation）— 理解重写**
动宾集群是领导喜欢材料的典型特征：由动词加宾语构成的简洁凝练的动宾短句连缀而成。

主客二分法：将一切可支配可影响的要素（而非静态背景或中介）视为能动主体可以主动施加动作的客体。

六层客体化：
- 战略与理念层：树立理念、贯彻思想、谋划蓝图（不是"以新理念为指导"）
- 目标与任务层：瞄准前沿、锁定难题、确立目标（不是"以打造什么为目标"）
- 资源与工具层：搭建平台、汇聚资源、用好工具、激活要素（不是"利用什么平台"）
- 方法与路径层：创新模式、探索路径、打通环节、优化流程（不是"通过什么方式"）
- 主体与力量层：动员力量、渐强队伍、激发主体活力（不是"依靠什么力量"）
- 环境与生态层：构建生态、塑造环境、破除壁垒（不是"依托什么环境"）

**A1（past case）**
传统表达（太绕）：
"以新型研发机构为依托，充分发挥其优势作用，通过什么模式什么方式，围绕人工智能前沿领域..."
动宾集群表达：
"充分发挥新型研发机构优势（主体与力量层）持续探索可持续发展模式（方法与路径层）聚焦人工智能前沿领域（目标与任务层）组织各类研发主体开展联合攻关（资源与工具层）"

**A2（future trigger）**
你要写一段关于"科技创新"的领导讲话，需要用动宾集群体现行动密集推进的感觉。

**E（Execution）— 执行步骤**
1. **识别要素**：找出工作中涉及的所有要素（战略理念/目标任务/资源工具/方法路径/主体力量/环境生态）
2. **主客二分**：不要将任何要素当作静态背景或中介，而是当作需要被主动作用的客体
3. **动词选择**：选择有力的动词（树立/贯彻/谋划/瞄准/锁定/确立/搭建/汇聚/用好/创新/探索/打通）
4. **宾语匹配**：确保动词与宾语形成简洁有力的动宾结构
5. **连缀成群**：多个动宾短句连缀，形成动宾集群

**B（Boundary）— 边界与局限**
- **失效场景**：需要详细解释工作背景和逻辑链的场合
- **常见错误**：过度追求动宾结构导致表述生硬、不自然
- **作者盲点**：动宾集群是结果不是目的，核心是主客二分的思维方式
