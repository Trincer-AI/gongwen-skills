# 核心功能驱动框架

**R（Reading）— 原文引用**
> 区县主官汇报交流发言的核心功能驱动框架：战略定位→核心功能→具体举措→成效数据

**I（Interpretation）— 理解重写**
核心功能驱动框架是区县主官在高层级会议（如市季度经济运行分析会）上汇报交流发言的典型逻辑框架。

核心功能驱动框架的四梁八柱：
- **战略定位**：明确上级战略部署和本区域的对应使命（材料的魂）
- **核心功能**：将宏观战略分解为几项具体可衡量的核心功能（材料的四梁八柱）
- **具体举措**：在每项核心功能下填充能直接支撑的工作举措（要素分解：主体/载体/通道/环境/效能）
- **成效数据**：用直观的经济数据、项目进展说明功能得到增强

找核心功能的三种路径：
- **自上而下**：从权威表述中对标对表（上级战略规划与领导讲话）
- **自下而上**：从家底优势中提炼长板（区域优势与资源禀赋）
- **由内而外**：从区域竞争中差异定位（寻找差异化赛道）

**A1（past case）**
"增强门户枢纽功能"的核心功能驱动框架：
- 【主体】引进DHL、中国外运等头部物流企业设立区域总部
- 【载体】枢纽港产业园（核心空间载体）、西部陆海新通道运营中心（功能平台载体）
- 【通道】开通至东盟千里青州货运航线（物理通道）、铁供水多式联运枢纽（衔接通道）
- 【环境】深化通关一体化改革（制度环境）、创设专业化帮办代办公司（服务环境）
- 【效能】某某港货物吞吐量增长23.8%、进出口总额增长94.7%

**A2（future trigger）**
你要为某区区长写一份在全市高质量发展推进会上的汇报交流发言，需要突出该区在全市发展格局中的战略定位和核心功能。

**E（Execution）— 执行步骤**
1. **明确战略定位**：梳理本区域在全市一盘棋中扮演什么角色，发展如何支撑全市整体战略
2. **锁定核心功能**：自上而下（对标上级规划）/ 自下而上（从家底优势提炼）/ 由内而外（区域竞争差异定位）
3. **填充具体举措**：用要素分解法（主体/载体/通道/环境/效能）为每项核心功能配套工作举措
4. **匹配成效数据**：用直观的经济数据证明核心功能得到增强

**B（Boundary）— 边界与局限**
- **失效场景**：不是高层级汇报场合、不需要突出战略定位时不必用此框架
- **常见错误**：核心功能写得过于宏观（需要具体可衡量）、具体举措与核心功能不对应（需要逻辑关联）
- **作者盲点**：有时核心功能找起来困难，三种路径可以结合使用
